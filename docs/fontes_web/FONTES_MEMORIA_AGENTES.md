## Notas consolidadas (sintese)
- O modelo de memoria e hierarquico e combina memoria de organizacao, projeto e usuario.
- Arquivos de memoria podem ser carregados automaticamente conforme o contexto do diretorio.
- O comando `/memory` permite inspecionar/editar memorias em sessao.
- O atalho `#` acelera captura de novas instrucoes operacionais.
- A estrategia recomendada e manter instrucoes especificas, estruturadas e revisadas periodicamente.
- Arquivos de memoria podem importar outros arquivos por caminho (`@arquivo`) com limite de profundidade.

## Mapeamento para este repositorio
- `codex.md` => regra mestra operacional.
- `MEMORY.md` => consolidacao de aprendizados.
- `ERRORS.md` => historico de erros e prevencao.
- `CODEX.local.md` => preferencias locais nao versionadas.
